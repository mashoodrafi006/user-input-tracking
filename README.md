# Souqalmal

## Get Started

1. **Install [Node] version v10.13.0,  install mongodb version v4.4.4**
2. **Clone this repository.** - `git clone https://github.com/mashoodrafi006/Souqalmal.git`
3. **Install Node Packages.** - `npm install`
